# 技術スタック

Atlas Note の採用予定技術をまとめます。実装開始後、実際の依存関係に合わせて更新してください。

| カテゴリ | 採用 |
| --- | --- |
| Desktop | Wails |
| Backend | Go |
| Frontend | Vue 3 |
| Language | Go + TypeScript |
| Build | Vite |
| Styling | UnoCSS |
| UI | Reka UI |
| State | Composables + Pinia |
| Database | SQLite |
| Editor | Tiptap + CodeMirror |
| Storage | Markdown |
| Data Access | Repository + Squirrel |
| Sync | WebDAV を中心に検討 |
| AI | ユーザー自身の API Key を利用する方針 |
