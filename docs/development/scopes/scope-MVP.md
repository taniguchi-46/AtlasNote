# MVP（v0.1）詳細スコープ

## 目的
毎日使えるローカルメモアプリを完成させる。

## 開発方針
- `dev` ブランチをベースとし、Phaseごとのブランチ（例: `phase/mvp`）を作成する。
- 各機能は1機能実装ブランチを切り、完了時にPhaseブランチへマージする。

## 実装機能要件

### 基本機能
- メモ作成
- メモ編集
- メモ削除
- 自動保存
- メモ同期UI（実同期はPhase 3）
- メモ検索UI（検索本体はPhase 2）
- ノート一覧
- ノートブック（階層構造）
- ノートブックアイコン
  - デフォルトアイコンを複数用意
  - ユーザー追加アイコンに対応
- ゴミ箱
- お気に入り
- ピン留め
- ウィンドウを常に最前面に配置

### エディタ
- Markdown
- WYSIWYG
- リッチテキスト
- コードブロック
- 表
- チェックリスト
- 引用
- リンク
- 表操作改善
  - `3 x 3` の表挿入
  - 行追加
  - 列追加
  - テーブル削除
  - 列削除
  - 行削除
  - `Ctrl + Enter` による新規行追加
- 表の中に表を作れてしまう不具合の修正

### UI
- 3ペインレイアウト
- Phase 3までの主要機能の先行UI実装
- メイン画面上部バー
  - メモ同期
  - メモ検索
  - 新しいノート
  - ウィンドウを常に最前面に配置
  - 設定
- 右側ノート部分の上部プルダウンメニュー削除
- ライトテーマ
- ダークテーマ
- レスポンシブ対応（最低限）

### 設定画面
- フォント設定
  - デフォルト: Meiryo
- テーマ
- 一般
  - 既定のノートブックアイコン
- エディター
  - タイポグラフィ
  - 新規ノート1行目のスタイル
  - 行の長さ、行間、段落間隔

### 保存基盤
- SQLite (メタデータ: ノート情報、ノートブック構造、お気に入り、ピン留め、削除フラグ等の管理)
- Markdown保存 (ノート本文の保存)

## 対象外機能 (今後のPhase)
- タグ管理、各種検索機能、フィルターの実処理、テーブルコピー (Phase 2)
- WebDAV等の同期実処理 (Phase 3)
- AI関連機能 (Phase 4, 5)
- 外部連携やエクスポート (Phase 6, 7)
- 画像貼り付け、ドラッグ＆ドロップ、添付ファイル保存
- 自動バックアップ、バックアップ復元
- グローバルショートカット
