# MVP（v0.1）詳細スコープ

## 目的
毎日使えるローカルメモアプリを完成させる。

## 開発方針
- `dev` ブランチをベースとし、Phaseごとのブランチ（例: `phase/mvp`）を作成する。
- 各機能は1機能実装ブランチを切り、完了時にPhaseブランチへマージする。

## 実装機能要件

### 基本機能
- メモ作成
- メモ編集
- メモ削除
- 自動保存
- ノート一覧
- ノートブック（階層構造）
- ゴミ箱
- お気に入り
- ピン留め

### エディタ
- Markdown
- WYSIWYG
- リッチテキスト
- コードブロック
- 表
- チェックリスト
- 引用
- リンク
- 画像貼り付け
- ドラッグ＆ドロップ

### UI
- 3ペインレイアウト
- ライトテーマ
- ダークテーマ
- レスポンシブ対応（最低限）

### 保存基盤
- SQLite (メタデータ: ノート情報、ノートブック構造、お気に入り、ピン留め、削除フラグ等の管理)
- Markdown保存 (ノート本文の保存)

## 対象外機能 (今後のPhase)
- タグ管理、各種検索機能、フィルター (Phase 2)
- WebDAV等の同期機能 (Phase 3)
- AI関連機能 (Phase 4, 5)
- 外部連携やエクスポート (Phase 6, 7)
